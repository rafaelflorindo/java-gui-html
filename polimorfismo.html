Polimorfismo

Polimorfismo é um dos pilares da Programação Orientada a Objetos. A palavra significa "muitas formas".

Na prática, o polimorfismo permite que um mesmo método tenha comportamentos diferentes dependendo da classe que o implementa.

Isso ocorre geralmente quando utilizamos herança e sobrescrita de métodos.

Classe base

Primeiro criamos uma classe genérica chamada Animal. 


class Animal{

    public void emitirSom(){
        System.out.println("Som do animal");
    }

}

Classe derivada

Agora criamos uma classe Cachorro que herda da classe Animal e sobrescreve o método emitirSom(). 


class Cachorro extends Animal{

    public void emitirSom(){
        System.out.println("Latido");
    }

}

Exemplo completo


class Animal{

    public void emitirSom(){
        System.out.println("Som do animal");
    }

}

class Cachorro extends Animal{

    public void emitirSom(){
        System.out.println("Latido");
    }

}

public class Teste{

    public static void main(String[] args){

        Animal a1 = new Animal();
        Animal a2 = new Cachorro();

        a1.emitirSom();
        a2.emitirSom();

    }

}

Explicação

Observe a linha abaixo: 


Animal a2 = new Cachorro();

A variável é do tipo Animal, mas o objeto criado é do tipo Cachorro.

Quando o método emitirSom() é chamado, o Java executa a versão do método da classe Cachorro.

Isso é o que chamamos de polimorfismo.

Outro exemplo

Podemos criar outras classes que também herdam de Animal. 


class Gato extends Animal{

    public void emitirSom(){
        System.out.println("Miado");
    }

}

Agora diferentes objetos podem ter comportamentos diferentes utilizando o mesmo método.

Resumo

  • Polimorfismo significa "muitas formas"
  • Permite que métodos tenham comportamentos diferentes
  • Normalmente ocorre com herança
  • Utiliza sobrescrita de métodos

Atividade

Crie um programa com a classe Veiculo contendo o método mover().

Depois crie duas classes que herdam de Veiculo:

  • Carro
  • Moto

Cada classe deve sobrescrever o método mover() exibindo uma mensagem diferente.
