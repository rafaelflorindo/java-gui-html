Polimorfismo

Polimorfismo permite que um método tenha diferentes comportamentos dependendo da classe que o implementa.

Exemplo


class Animal{

    public void emitirSom(){
        System.out.println("Som do animal");
    }

}

class Cachorro extends Animal{

    public void emitirSom(){
        System.out.println("Latido");
    }

}
