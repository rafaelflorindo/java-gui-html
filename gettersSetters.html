Getters e Setters

Getters e Setters são métodos utilizados para acessar e modificar atributos privados.

Exemplo


public class Pessoa {

    private String nome;

    public String getNome(){
        return nome;
    }

    public void setNome(String nome){
        this.nome = nome;
    }

}

Isso garante maior controle sobre os dados.
