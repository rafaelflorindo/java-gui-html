Encapsulamento

Encapsulamento é um dos pilares da Programação Orientada a Objetos. Ele consiste em proteger os atributos de uma classe, permitindo que eles sejam acessados ou modificados apenas através de métodos.

Isso ajuda a manter a segurança e a organização do código, evitando que outras partes do programa alterem dados de forma indevida.

Atributos privados

Para proteger os atributos utilizamos o modificador de acesso private.

Quando um atributo é declarado como private, ele só pode ser acessado dentro da própria classe.

Métodos getters e setters

Para acessar ou modificar os atributos privados, utilizamos métodos especiais chamados:

  • get → utilizado para obter o valor do atributo
  • set → utilizado para alterar o valor do atributo

Exemplo


public class Pessoa {

    private String nome;

    public String getNome(){
        return nome;
    }

    public void setNome(String nome){
        this.nome = nome;
    }

}

Explicação do código

  • private String nome → atributo protegido
  • getNome() → retorna o valor do atributo
  • setNome() → altera o valor do atributo

A palavra-chave this é utilizada para indicar que estamos acessando o atributo da própria classe.

Exemplo completo


class Pessoa {

    private String nome;

    public String getNome(){
        return nome;
    }

    public void setNome(String nome){
        this.nome = nome;
    }

}

public class Teste {

    public static void main(String[] args){

        Pessoa pessoa = new Pessoa();

        pessoa.setNome("Ana");

        System.out.println(pessoa.getNome());

    }

}

Explicação

Neste exemplo, não podemos acessar diretamente o atributo nome, pois ele é private.

Por isso utilizamos os métodos:

  • setNome() para definir o valor
  • getNome() para acessar o valor

Vantagens do Encapsulamento

  • Protege os dados da classe
  • Evita alterações indevidas
  • Melhora a organização do código
  • Facilita a manutenção do sistema

Atividade

Crie uma classe chamada Produto com os seguintes atributos:

  • nome
  • preco

Os atributos devem ser private.

Crie métodos get e set para cada atributo e depois crie um objeto da classe para testar o funcionamento.
