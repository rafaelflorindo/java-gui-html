Encapsulamento

Encapsulamento é o princípio que protege os atributos da classe, permitindo acesso apenas através de métodos.

Exemplo


public class Pessoa {

    private String nome;

    public String getNome(){
        return nome;
    }

    public void setNome(String nome){
        this.nome = nome;
    }

}
