Validação de Dados

Durante o desenvolvimento de programas, é fundamental garantir que os dados informados pelo usuário sejam válidos. Isso evita erros e melhora a confiabilidade do sistema.

Validação na entrada de dados

Podemos utilizar estruturas de repetição para garantir que o usuário informe valores válidos. 


int idade;

do {
    System.out.print("Digite uma idade válida: ");
    idade = scanner.nextInt();
} while (idade < 0);

Nesse exemplo, o programa continuará solicitando a idade até que o valor seja maior ou igual a zero.

Validação em Programação Orientada a Objetos

Além de validar a entrada de dados, também é importante garantir que os objetos do sistema nunca armazenem valores inválidos.

Para isso, utilizamos os métodos setters. 


public class Pessoa {

    private int idade;

    public void setIdade(int idade) {

        if (idade < 0) {
            System.out.println("Idade inválida!");
        } else {
            this.idade = idade;
        }
    }

    public int getIdade() {
        return idade;
    }
}

Utilizando as duas validações

O ideal é utilizar as duas formas de validação:

  • Validação na entrada → evita erros no programa
  • Validação no setter → protege o objeto

import java.util.Scanner;

public class Programa {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);
        Pessoa p = new Pessoa();

        int idade;

        do {
            System.out.print("Digite uma idade válida: ");
            idade = scanner.nextInt();
        } while (idade < 0);

        p.setIdade(idade);

        System.out.println("Idade cadastrada: " + p.getIdade());

        scanner.close();
    }
}

Boas práticas

  • Sempre validar dados informados pelo usuário
  • Não confiar em entradas externas
  • Utilizar setters para proteger os objetos
  • Evitar que o sistema trabalhe com dados inválidos

Atividade

Crie um programa que:

  • Solicite a idade do usuário
  • Valide a entrada (não permitir valores negativos)
  • Utilize uma classe com setter para armazenar a idade
  • Exiba o valor ao final
