Herança

Herança é um mecanismo da Programação Orientada a Objetos que permite que uma classe herde atributos e métodos de outra classe.

A classe que fornece as características é chamada de classe pai (ou superclasse), enquanto a classe que herda essas características é chamada de classe filha (ou subclasse).

Em Java utilizamos a palavra-chave extends para indicar que uma classe está herdando de outra.

Exemplo simples


public class Pessoa {

    String nome;

}

public class Aluno extends Pessoa {

    int matricula;

}

Explicação

  • A classe Pessoa possui o atributo nome
  • A classe Aluno herda esse atributo
  • Além disso, Aluno possui seu próprio atributo chamado matricula

Isso significa que um objeto da classe Aluno terá acesso tanto ao atributo nome quanto ao atributo matricula.

Exemplo completo


class Pessoa {

    String nome;

    public void apresentar(){
        System.out.println("Meu nome é " + nome);
    }

}

class Aluno extends Pessoa {

    int matricula;

}

public class Teste {

    public static void main(String[] args){

        Aluno aluno = new Aluno();

        aluno.nome = "Carlos";
        aluno.matricula = 123;

        aluno.apresentar();

    }

}

Explicação do exemplo

Neste exemplo, a classe Aluno herda o atributo nome e o método apresentar() da classe Pessoa.

Assim, mesmo que esses elementos não estejam declarados diretamente na classe Aluno, eles podem ser utilizados normalmente.

Vantagens da Herança

  • Reutilização de código
  • Organização das classes
  • Facilidade de manutenção
  • Possibilita o uso de polimorfismo

Atividade

Crie uma classe chamada Funcionario contendo:

  • nome
  • salario

Depois crie uma classe chamada Gerente que herda de Funcionario e adicione um atributo chamado departamento.

Crie um objeto da classe Gerente e exiba suas informações.
