Arrays em Java

Um Array é uma estrutura utilizada para armazenar vários valores do mesmo tipo em uma única variável.

Cada valor é armazenado em uma posição chamada índice. Os índices começam sempre em 0.

Criando um Array


int[] numeros = new int[3];

Neste exemplo criamos um array chamado numeros com espaço para armazenar 3 números inteiros.

Utilizando o Array sem laço de repetição


int[] numeros = new int[3];

numeros[0] = 10;
numeros[1] = 20;
numeros[2] = 30;

System.out.println(numeros[0]);
System.out.println(numeros[1]);
System.out.println(numeros[2]);

Cada posição do array é acessada utilizando o índice.

Utilizando o Array com laço de repetição

Quando precisamos acessar todos os elementos do array, normalmente utilizamos um laço for. 


int[] numeros = {10, 20, 30};

for(int i = 0; i < numeros.length; i++){

    System.out.println(numeros[i]);

}

O atributo length indica o tamanho do array.

Arrays de Objetos

Arrays também podem armazenar objetos. Para isso, precisamos primeiro criar uma classe.

Classe Pessoa


public class Pessoa {

    private int id;
    private String nome;

    public Pessoa(int id, String nome){
        this.id = id;
        this.nome = nome;
    }

    public int getId(){
        return id;
    }

    public String getNome(){
        return nome;
    }

}

Essa classe possui dois atributos:

  • id
  • nome

Criando um Array de Pessoa


Pessoa[] pessoas = new Pessoa[3];

pessoas[0] = new Pessoa(1, "Ana");
pessoas[1] = new Pessoa(2, "Carlos");
pessoas[2] = new Pessoa(3, "Maria");

Percorrendo o Array de Pessoas


for(int i = 0; i < pessoas.length; i++){

    System.out.println(
        pessoas[i].getId() + " - " + pessoas[i].getNome()
    );

}

Neste exemplo utilizamos o método getId() e getNome() para acessar os dados do objeto.

Resumo

  • Arrays armazenam vários valores do mesmo tipo
  • O índice sempre começa em 0
  • O tamanho do array é fixo
  • Arrays também podem armazenar objetos

Atividade

Crie um array com 4 objetos Pessoa e exiba os dados utilizando um for.
