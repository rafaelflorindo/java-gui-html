Métodos em Java

Métodos são blocos de código responsáveis por executar uma determinada ação. Eles ajudam a organizar o programa e evitar repetição de código.

Estrutura de um método


tipoRetorno nomeMetodo(parametros){
    // código
}

Exemplo simples


public class Exemplo {

    public static void main(String[] args) {
        saudacao();
    }

    public static void saudacao(){
        System.out.println("Olá!");
    }

}

Método com parâmetro


public static void mostrarNome(String nome){
    System.out.println("Nome: " + nome);
}

Método com retorno


public static int somar(int a, int b){
    return a + b;
}

O return devolve um valor ao método que fez a chamada.

Boas práticas

  • Utilizar nomes claros
  • Evitar métodos muito grandes
  • Cada método deve ter uma responsabilidade
