Classes em Java

Uma classe é um modelo ou estrutura utilizada para criar objetos. Ela define quais atributos (características) e métodos (ações) os objetos terão.

Na Programação Orientada a Objetos, uma classe funciona como um molde para criar vários objetos semelhantes.

Atributos

Os atributos representam as características de um objeto.

Exemplo:


public class Pessoa {

    String nome;
    int idade;

}

Neste exemplo, a classe Pessoa possui dois atributos:

  • nome
  • idade

Métodos

Os métodos representam as ações que um objeto pode realizar.

Exemplo:


public class Pessoa {

    String nome;
    int idade;

    public void apresentar(){
        System.out.println("Meu nome é " + nome);
    }

}

O método apresentar() exibe uma mensagem no console.

Criando objetos

Um objeto é uma instância de uma classe. Para criar um objeto utilizamos a palavra-chave new. 


public class Teste {

    public static void main(String[] args){

        Pessoa pessoa = new Pessoa();

        pessoa.nome = "Carlos";
        pessoa.idade = 25;

        pessoa.apresentar();

    }

}

Explicação do exemplo

  • Pessoa pessoa = new Pessoa() → cria um objeto da classe Pessoa
  • pessoa.nome → define o nome
  • pessoa.idade → define a idade
  • pessoa.apresentar() → executa o método da classe

Resumo

  • Classe → modelo para criar objetos
  • Atributos → características do objeto
  • Métodos → ações que o objeto pode executar
  • Objeto → instância de uma classe
