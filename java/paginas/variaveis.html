Variáveis em Java

Variáveis são utilizadas para armazenar dados durante a execução de um programa. Cada variável possui um nome, um tipo de dado e um valor armazenado.

Em Java, antes de utilizar uma variável, é necessário declará-la informando o tipo de dado que ela irá armazenar.

Estrutura de uma variável


tipo nomeDaVariavel = valor;

Exemplo:


int idade = 20;
  • int → tipo da variável
  • idade → nome da variável
  • 20 → valor armazenado

Tipos de dados em Java

Em Java, os tipos de dados são divididos em duas categorias principais:

Tipos Primitivos

São tipos básicos que armazenam valores diretamente na memória.

  • int → números inteiros
  • double → números decimais
  • char → um único caractere
  • boolean → verdadeiro ou falso

Tipos de Referência (Não Primitivos)

São tipos que armazenam uma referência para um objeto na memória.

  • String → textos
  • Arrays
  • Objetos (Classes)

Os tipos de referência serão estudados em detalhes em seções específicas, onde veremos como trabalhar com arrays e classes.

Exemplo de uso de variáveis


public class Exemplo {

    public static void main(String[] args) {

        int idade = 20;
        double salario = 1500.50;
        String nome = "Maria";

        System.out.println(nome);

    }

}

Explicação do código

Neste exemplo foram declaradas três variáveis:

  • idade → armazena um número inteiro (tipo primitivo)
  • salario → armazena um número decimal (tipo primitivo)
  • nome → armazena um texto (tipo de referência)

O comando abaixo exibe o valor da variável nome no console: 


System.out.println(nome);

Outro exemplo


public class DadosAluno {

    public static void main(String[] args) {

        String nome = "Carlos";
        int idade = 22;
        double nota = 8.5;

        System.out.println("Aluno: " + nome);
        System.out.println("Idade: " + idade);
        System.out.println("Nota: " + nota);

    }

}

Boas práticas ao nomear variáveis

  • Utilizar nomes que representem o conteúdo da variável
  • Evitar nomes muito curtos
  • Utilizar padrão camelCase

Exemplos:


int idadeAluno;
double valorProduto;
String nomeCliente;
