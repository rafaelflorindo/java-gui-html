Introdução ao Swing (via código)

Até agora utilizamos o console para interagir com o usuário.

A partir deste momento, vamos utilizar o Swing, uma biblioteca do Java para criação de interfaces gráficas.

Objetivo

Criar uma tela que:

  • Recebe nome e idade
  • Ao clicar no botão, exibe os dados na tela

Exemplo completo


import javax.swing.*;
import java.awt.event.*;

public class Tela {

    public static void main(String[] args){

        JFrame janela = new JFrame("Cadastro");

        JLabel lblNome = new JLabel("Nome:");
        lblNome.setBounds(20, 20, 100, 25);

        JTextField txtNome = new JTextField();
        txtNome.setBounds(100, 20, 150, 25);

        JLabel lblIdade = new JLabel("Idade:");
        lblIdade.setBounds(20, 60, 100, 25);

        JTextField txtIdade = new JTextField();
        txtIdade.setBounds(100, 60, 150, 25);

        JButton botao = new JButton("Enviar");
        botao.setBounds(100, 100, 100, 30);

        JLabel lblResultado = new JLabel("");
        lblResultado.setBounds(20, 150, 300, 25);

        botao.addActionListener(new ActionListener(){
            public void actionPerformed(ActionEvent e){

                String nome = txtNome.getText();

                // Conversão de String para int
                int idade = Integer.parseInt(txtIdade.getText());

                // Convertendo novamente para String
                String resultado = "Nome: " + nome + " | Idade: " + idade;

                lblResultado.setText(resultado);
            }
        });

        janela.add(lblNome);
        janela.add(txtNome);
        janela.add(lblIdade);
        janela.add(txtIdade);
        janela.add(botao);
        janela.add(lblResultado);

        janela.setSize(350, 250);
        janela.setLayout(null);
        janela.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);
        janela.setVisible(true);
    }
}

Explicação

Componentes utilizados

  • JFrame → janela principal
  • JLabel → texto na tela
  • JTextField → campo de entrada
  • JButton → botão

Posicionamento

Utilizamos o método setBounds(x, y, largura, altura) para posicionar os componentes na tela.

ActionListener

O ActionListener é responsável por capturar o clique do botão.

Tudo que estiver dentro de actionPerformed será executado ao clicar.

Conversão de dados

O JTextField sempre retorna String. 


int idade = Integer.parseInt(txtIdade.getText());

Aqui convertemos para int.

Depois, ao exibir, o Java automaticamente converte para String.

Importante

  • Interface via código é mais trabalhosa
  • Serve para entender o funcionamento
  • Depois utilizaremos interface gráfica da IDE
