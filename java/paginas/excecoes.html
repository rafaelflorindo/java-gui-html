Tratamento de Exceções

Durante a execução de um programa, podem ocorrer erros inesperados, como entrada de dados inválida ou operações impossíveis de serem realizadas.

Esses erros são chamados de exceções. Em Java, podemos tratar essas situações para evitar que o programa seja interrompido.

Exemplo de erro

Observe o código abaixo: 


Scanner scanner = new Scanner(System.in);

System.out.print("Digite um número: ");
int numero = scanner.nextInt();

Se o usuário digitar texto ao invés de um número, o programa será interrompido com erro.

Tratando erros com try e catch

Podemos utilizar o bloco try/catch para capturar e tratar esse erro. 


import java.util.Scanner;

public class Exemplo {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        try {
            System.out.print("Digite um número: ");
            int numero = scanner.nextInt();

            System.out.println("Número digitado: " + numero);

        } catch (Exception e) {
            System.out.println("Erro: valor inválido!");
        }

        scanner.close();
    }
}

Explicação

  • try → bloco onde pode ocorrer erro
  • catch → trata o erro ocorrido
  • Exception → tipo genérico de erro

Exibindo a mensagem do erro


catch (Exception e) {
    System.out.println(e.getMessage());
}

O método getMessage() retorna a mensagem original do erro.

Exemplo com repetição

Podemos combinar tratamento de exceções com estruturas de repetição: 


int numero = 0;
boolean valido = false;

while (!valido) {
    try {
        System.out.print("Digite um número inteiro: ");
        numero = scanner.nextInt();
        valido = true;
    } catch (Exception e) {
        System.out.println("Valor inválido! Tente novamente.");
        scanner.next(); // limpa entrada inválida
    }
}

Exceções em Programação Orientada a Objetos

Também podemos utilizar exceções para garantir regras dentro das classes. 


public class Pessoa {

    private int idade;

    public void setIdade(int idade) {

        if (idade < 0) {
            throw new IllegalArgumentException("Idade não pode ser negativa");
        }

        this.idade = idade;
    }

    public int getIdade() {
        return idade;
    }
}

Utilizando a exceção


try {
    p.setIdade(-10);
} catch (Exception e) {
    System.out.println(e.getMessage());
}

Boas práticas

  • Utilizar try/catch para evitar que o programa quebre
  • Exibir mensagens claras para o usuário
  • Evitar capturar exceções genéricas em excesso
  • Utilizar exceções para proteger regras do sistema

Atividade

Crie um programa que:

  • Solicite um número inteiro ao usuário
  • Utilize try/catch para tratar entradas inválidas
  • Repita a leitura até o valor ser válido
  • Exiba o número ao final

Desafio

Crie uma classe com um setter que lance uma exceção caso o valor seja inválido, e trate essa exceção no método principal.
