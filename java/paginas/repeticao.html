Estruturas de Repetição em Java

As estruturas de repetição permitem executar um bloco de código várias vezes, de acordo com uma condição definida pelo programador.

Elas são muito utilizadas quando precisamos repetir tarefas, como percorrer listas, exibir valores ou realizar cálculos repetitivos.

Principais estruturas de repetição

  • for
  • while
  • do while

Estrutura for

A estrutura for é utilizada quando sabemos antecipadamente quantas vezes o bloco de código deverá ser executado.

Sintaxe:


for(inicialização; condição; incremento){
    // código que será repetido
}

Exemplo com for


public class Exemplo {

    public static void main(String[] args) {

        for(int i = 1; i <= 5; i++){

            System.out.println(i);

        }

    }

}

Explicação do código

  • int i = 1 → inicializa a variável de controle
  • i <= 5 → condição para continuar o loop
  • i++ → incrementa o valor da variável

Nesse exemplo, o programa exibirá os números de 1 até 5.

Estrutura while

A estrutura while é utilizada quando não sabemos exatamente quantas vezes o código será executado, mas temos uma condição que controla a repetição.

Sintaxe:


while(condição){
    // código executado enquanto a condição for verdadeira
}

Exemplo com while


int contador = 1;

while(contador <= 5){

    System.out.println(contador);

    contador++;

}

Neste exemplo, o programa exibirá os números de 1 até 5 enquanto a condição contador <= 5 for verdadeira.

Estrutura do while

A estrutura do while é semelhante ao while, porém possui uma diferença importante: o bloco de código é executado pelo menos uma vez antes da verificação da condição.

Sintaxe:


do{
    // código executado
}while(condição);

Exemplo com do while


int numero = 1;

do{

    System.out.println(numero);

    numero++;

}while(numero <= 5);

Nesse caso, o bloco será executado primeiro e somente depois a condição será verificada.

Comparação entre as estruturas

  • for → usado quando sabemos o número de repetições
  • while → usado quando a repetição depende de uma condição
  • do while → garante pelo menos uma execução

Exemplo aplicado

O programa abaixo exibe a tabuada do número 5: 


public class Tabuada {

    public static void main(String[] args) {

        for(int i = 1; i <= 10; i++){

            System.out.println("5 x " + i + " = " + (5 * i));

        }

    }

}

Atividade

Crie um programa em Java que utilize uma estrutura de repetição para exibir os números de 1 até 10.

Depois modifique o programa para exibir apenas os números pares.
