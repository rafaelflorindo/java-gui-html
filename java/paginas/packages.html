Packages em Java

Packages (pacotes) são utilizados para organizar classes em Java.

Eles funcionam como pastas dentro do projeto, permitindo separar o código por responsabilidade.

Por que usar packages?

  • Organizar melhor o projeto
  • Facilitar a manutenção
  • Evitar conflitos de nomes

Exemplo de declaração


package model;

Essa linha deve estar no início do arquivo Java.

Organizando as classes

Uma boa prática é separar as classes por tipo:

  • model → classes de dados (ex: Pessoa)
  • controller → regras do sistema
  • dao → acesso a dados

Exemplo de estrutura


src/
 ├── model/
 │    └── Pessoa.java
 ├── controller/
 │    └── PessoaController.java
 ├── dao/
 │    └── PessoaDAO.java
 └── Main.java

Importando classes de outro package


import model.Pessoa;

Isso permite utilizar a classe em outro arquivo.

Exemplo prático

Classe no package model: 


package model;

public class Pessoa {

    private String nome;

    public Pessoa(String nome){
        this.nome = nome;
    }

}

Classe principal utilizando a classe Pessoa: 


import model.Pessoa;

public class Main {

    public static void main(String[] args) {

        Pessoa p = new Pessoa("Maria");

    }

}

Boas práticas

  • Utilizar nomes simples e em minúsculo
  • Separar classes por responsabilidade
  • Evitar deixar todas as classes no mesmo pacote
