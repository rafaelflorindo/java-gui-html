Busca em ArrayList

Buscar dados em uma lista é uma das operações mais importantes em programação.

A busca consiste em percorrer os elementos e verificar se algum deles atende a uma determinada condição.

Busca simples em lista de String


ArrayList nomes = new ArrayList<>();

nomes.add("Ana");
nomes.add("Carlos");
nomes.add("Maria");

String busca = "Maria";

for(String nome : nomes){

    if(nome.equals(busca)){
        System.out.println("Nome encontrado!");
    }

}

Neste exemplo, percorremos toda a lista comparando cada elemento com o valor buscado.

Utilizando variável de controle

É comum utilizar uma variável para indicar se o elemento foi encontrado. 


boolean encontrado = false;

for(String nome : nomes){

    if(nome.equals(busca)){
        encontrado = true;
        break;
    }

}

if(encontrado){
    System.out.println("Encontrado!");
}else{
    System.out.println("Não encontrado!");
}

Por que usar break?

O break interrompe o laço assim que o elemento é encontrado, evitando processamento desnecessário.

Busca em lista de objetos

Também é possível realizar buscas em listas de objetos. 


for(Pessoa p : pessoas){

    if(p.getNome().equals(busca)){
        System.out.println("Pessoa encontrada: " + p.getNome());
    }

}

Neste caso, utilizamos o método getNome() para acessar o atributo.

Busca por ID


int idBusca = 2;

for(Pessoa p : pessoas){

    if(p.getId() == idBusca){
        System.out.println("Pessoa encontrada: " + p.getNome());
    }

}

Boas práticas

  • Utilizar equals() para comparar textos
  • Utilizar break quando não precisar continuar a busca
  • Usar variável de controle para melhorar a lógica

Atividade
