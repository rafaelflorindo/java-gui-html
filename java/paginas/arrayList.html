ArrayList em Java

O ArrayList é uma estrutura de dados utilizada para armazenar vários elementos em uma lista.

Diferente dos arrays, o ArrayList possui tamanho dinâmico, ou seja, ele pode crescer conforme novos elementos são adicionados.

O ArrayList faz parte do conjunto de estruturas chamado Collections (Coleções) da linguagem Java. Neste material estamos utilizando apenas o ArrayList, mas em breve estudaremos outras estruturas que fazem parte das coleções.

Importando a classe

Antes de utilizar o ArrayList é necessário importar a classe. 


import java.util.ArrayList;

Criando um ArrayList


ArrayList nomes = new ArrayList<>();

Neste exemplo criamos uma lista chamada nomes que irá armazenar valores do tipo String.

Utilizando o ArrayList sem laço de repetição


ArrayList nomes = new ArrayList<>();

nomes.add("Ana");
nomes.add("Carlos");
nomes.add("Maria");

System.out.println(nomes.get(0));
System.out.println(nomes.get(1));
System.out.println(nomes.get(2));

Utilizamos:

  • add() para adicionar elementos
  • get() para acessar elementos

Utilizando o ArrayList com laço de repetição


ArrayList nomes = new ArrayList<>();

nomes.add("Ana");
nomes.add("Carlos");
nomes.add("Maria");

for(int i = 0; i < nomes.size(); i++){

    System.out.println(nomes.get(i));

}

Utilizamos o método size() para saber a quantidade de elementos armazenados na lista.

ArrayList de Objetos

Assim como os arrays, o ArrayList também pode armazenar objetos.

Para isso precisamos primeiro criar uma classe.

Classe Pessoa


public class Pessoa {

    private int id;
    private String nome;

    public Pessoa(int id, String nome){
        this.id = id;
        this.nome = nome;
    }

    public int getId(){
        return id;
    }

    public String getNome(){
        return nome;
    }

}

Criando um ArrayList de Pessoa


ArrayList pessoas = new ArrayList<>();

pessoas.add(new Pessoa(1, "Ana"));
pessoas.add(new Pessoa(2, "Carlos"));
pessoas.add(new Pessoa(3, "Maria"));

Percorrendo o ArrayList de Pessoas


for(int i = 0; i < pessoas.size(); i++){

    Pessoa p = pessoas.get(i);

    System.out.println(
        p.getId() + " - " + p.getNome()
    );

}

Alterando e removendo elementos


ArrayList nomes = new ArrayList<>();

nomes.add("Ana");
nomes.add("Carlos");
nomes.add("Maria");

// Alterando um elemento
nomes.set(1, "João");

// Removendo um elemento
nomes.remove(0);

// Exibindo a lista atualizada
for(int i = 0; i < nomes.size(); i++){
    System.out.println(nomes.get(i));
}

Utilizamos:

  • set() para alterar um elemento em uma posição específica
  • remove() para remover um elemento da lista

Após remover um elemento, os índices da lista são reorganizados automaticamente.

Resumo

  • ArrayList faz parte das Collections
  • Possui tamanho dinâmico
  • Possui métodos como add(), get() e size()
  • Pode armazenar objetos

Atividade

Crie um ArrayList de Pessoa e adicione pelo menos 4 pessoas. Em seguida, utilize um laço for para exibir os dados.
