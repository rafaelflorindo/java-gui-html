Entrada de Dados com Scanner

Em Java, para receber dados do usuário pelo teclado, utilizamos a classe Scanner.

Ela permite ler diferentes tipos de dados, como números, textos e valores booleanos.

Importando a classe


import java.util.Scanner;

Criando o Scanner


Scanner scanner = new Scanner(System.in);

O System.in indica que os dados serão lidos do teclado.

Lendo diferentes tipos de dados


import java.util.Scanner;

public class EntradaDados {

    public static void main(String[] args) {

        Scanner scanner = new Scanner(System.in);

        System.out.print("Digite um número inteiro: ");
        int numeroInteiro = scanner.nextInt();

        System.out.print("Digite um número decimal: ");
        double numeroDecimal = scanner.nextDouble();

        System.out.print("Digite um caractere: ");
        char caractere = scanner.next().charAt(0);

        System.out.print("Digite verdadeiro ou falso: ");
        boolean valorBooleano = scanner.nextBoolean();

        scanner.nextLine(); // limpeza de buffer

        System.out.print("Digite um texto: ");
        String texto = scanner.nextLine();

        System.out.println("\n--- Dados informados ---");
        System.out.println("Inteiro: " + numeroInteiro);
        System.out.println("Decimal: " + numeroDecimal);
        System.out.println("Caractere: " + caractere);
        System.out.println("Booleano: " + valorBooleano);
        System.out.println("Texto: " + texto);

        scanner.close();
    }
}

Explicação dos métodos

  • nextInt() → lê números inteiros
  • nextDouble() → lê números decimais
  • next() → lê uma palavra
  • nextLine() → lê uma linha completa
  • nextBoolean() → lê verdadeiro ou falso

Atenção ao nextLine()

Após utilizar métodos como nextInt() ou nextDouble(), é necessário utilizar um nextLine() para limpar o buffer do teclado.

Caso isso não seja feito, o próximo nextLine() pode ser ignorado.

Boas práticas

  • Sempre informar ao usuário o que deve ser digitado
  • Utilizar mensagens claras no console
  • Fechar o Scanner com close()
