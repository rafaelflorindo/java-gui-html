Estruturas Condicionais em Java

As estruturas condicionais permitem que o programa tome decisões durante a execução. Dependendo do resultado de uma condição lógica, diferentes blocos de código podem ser executados.

Essas estruturas são muito utilizadas quando precisamos verificar situações como idade, notas, valores ou qualquer outra condição.

Operadores relacionais

Para criar condições utilizamos operadores relacionais:

  • == igual
  • != diferente
  • > maior que
  • < menor que
  • >= maior ou igual
  • <= menor ou igual

Exemplo com if

A estrutura if executa um bloco de código somente se a condição for verdadeira. 


public class Exemplo {

    public static void main(String[] args) {

        int idade = 18;

        if(idade >= 18){
            System.out.println("Maior de idade");
        }

    }

}

Nesse exemplo, a mensagem será exibida apenas se a idade for maior ou igual a 18.

Exemplo com if / else

A estrutura else permite executar um bloco de código quando a condição do if for falsa. 


int idade = 16;

if(idade >= 18){
    System.out.println("Maior de idade");
}else{
    System.out.println("Menor de idade");
}

Se a condição não for verdadeira, o programa executa o bloco de código dentro do else.

Exemplo com if / else if

Quando precisamos verificar várias condições, utilizamos else if. 


int nota = 7;

if(nota >= 7){
    System.out.println("Aprovado");
}else if(nota >= 5){
    System.out.println("Recuperação");
}else{
    System.out.println("Reprovado");
}

Nesse exemplo:

  • nota maior ou igual a 7 → Aprovado
  • nota maior ou igual a 5 → Recuperação
  • caso contrário → Reprovado

Estrutura switch

O switch é utilizado quando precisamos comparar uma variável com vários valores possíveis. 


int dia = 3;

switch(dia){

    case 1:
        System.out.println("Domingo");
        break;

    case 2:
        System.out.println("Segunda-feira");
        break;

    case 3:
        System.out.println("Terça-feira");
        break;

    default:
        System.out.println("Dia inválido");

}

A palavra break interrompe a execução do bloco dentro do switch.

Operadores lógicos

Também podemos combinar condições utilizando operadores lógicos:

  • && → E lógico
  • || → OU lógico
  • ! → NÃO lógico

Exemplo:


int idade = 20;
boolean possuiCarteira = true;

if(idade >= 18 && possuiCarteira){
    System.out.println("Pode dirigir");
}

Atividade

Crie um programa em Java que:

  • Declare uma variável chamada nota
  • Se a nota for maior ou igual a 7 → exibir "Aprovado"
  • Se a nota for maior ou igual a 5 → exibir "Recuperação"
  • Caso contrário → exibir "Reprovado"
