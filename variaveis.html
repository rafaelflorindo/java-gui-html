Variáveis em Java

Variáveis são utilizadas para armazenar dados em um programa.

Exemplo


public class Exemplo {

    public static void main(String[] args) {

        int idade = 20;
        double salario = 1500.50;
        String nome = "Maria";

        System.out.println(nome);

    }

}
