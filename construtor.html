Construtor em Java

O construtor é um método especial utilizado para inicializar objetos. Ele é chamado automaticamente ao criar um objeto.

Exemplo


public class Pessoa {

    private String nome;

    public Pessoa(String nome){
        this.nome = nome;
    }

}

Criando objeto


Pessoa p = new Pessoa("Maria");

O this é utilizado para referenciar o atributo da classe.
